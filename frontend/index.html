Privacy Policy

Local Execution

Forge runs entirely on your machine. No data is sent to external servers, except API calls to Groq for the language model.

API Keys

Your GROQ_API_KEY is stored locally in your .env file and never logged, stored, or transmitted to any third party.

Generated Code

All generated and executed code remains on your local machine. Forge does not back up or transmit what it creates.

No Tracking

Forge contains zero analytics, tracking, or telemetry of any kind.

Setup Guide

Welcome to Forge

This guide will get you up and running in under two minutes. You'll need a Groq API key — it's free.

Forge is a local dev agent: you describe tasks in plain English, it plans and executes them on your machine.

Add Your API Key

Create a .env file in the project root with your Groq key:

GROQ_API_KEY=your_key_here

Get a free key at console.groq.com →

You're Ready

  • Plan: Describe a task in plain English
  • Review: Inspect each step before running
  • Execute: Watch Forge write and run your code
  • History: Every completed task is saved locally

All code stays on your machine. Nothing leaves without your knowledge.
